h2. Você encontrou um bug no software?

Você pode reportar o bug ("file a bug report"):https://leap.se/code/projects/report-issues/issues/new?issue[tracker_id]=2 no bug tracker do projeto LEAP. Primeiro, [[procure por problemas similares na base de dados de bugs reportados => https://leap.se/code/search]] para ver se o bug já não foi reportado.

h2. Deseja sugerir uma nova feature para o Bitmask?

Você pode requisitar novas funcionalidades para o software: https://leap.se/code/projects/report-issues/issues/new?issue[tracker_id]=1 no bug tracker do projeto LEAP. primeiro, [[procure por sugestões semelhantes => https://leap.se/code/search]] para garantir que a funcionalidade não foi sugerida por outra pessoa.
